# 📊 Customer Churn Analysis

## 📌 Overview

Customer Churn Analysis is a data analytics project focused on identifying customer attrition patterns and uncovering the key factors that influence customer retention. The project combines Python, SQL, and Power BI to transform raw customer data into actionable business insights.

The workflow includes data cleaning, exploratory data analysis (EDA), visualization, SQL-based business analysis, dashboard development, and presentation of findings through reports and presentations.

---

## 🎯 Project Objectives

* Analyze customer behavior and churn trends.
* Identify factors contributing to customer loss.
* Discover high-risk customer segments.
* Generate business insights to improve customer retention.
* Build an interactive dashboard for decision-making.

---

## 📂 Dataset

The dataset contains customer-related information such as:

* Customer Demographics
* Senior Citizen Status
* Tenure
* Contract Type
* Internet Services
* Payment Method
* Monthly Charges
* Total Charges
* Customer Churn Status

**Target Variable:** `Churn`

## 🔍 Key Insights

* Month-to-month contract customers showed higher churn rates.
* Customers with shorter tenure were more likely to leave.
* Electronic check users exhibited increased churn behavior.
* Higher monthly charges were associated with greater churn risk.
* Long-term contract customers demonstrated stronger retention.

## 💡 Business Recommendations

* Encourage customers to switch to long-term contracts.
* Improve retention strategies for new customers.
* Offer personalized incentives to high-risk customers.
* Optimize pricing and service bundles.

## ▶️ How to Run

### Python Analysis

1. Clone the repository.

```bash
git clone <https://github.com/Mayankverma30/Customer-Churn-Analysis>
```

2. Install required libraries.

```bash
pip install pandas numpy matplotlib seaborn
```

3. Run the Jupyter notebooks for:

   * Data Cleaning
   * EDA
   * Visualization

### SQL Analysis

1. Import the dataset into MySQL.
2. Execute SQL queries from:

```sql
churn_analysis_queries.sql
```

### Power BI Dashboard

1. Open the `.pbix` file in Power BI Desktop.
2. Refresh data connections.
3. Explore interactive dashboard insights.
